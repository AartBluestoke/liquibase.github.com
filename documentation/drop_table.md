# Drop Table #

Drops an existing table.

## Sample ##

{% highlight xml %}
<dropTable tableName="person" schemaName="mySchema"/>
{% endhighlight %}

## Available Attributes ##

^ tableName  | Name of the table to drop **required**  |
^ schemaName  | Name of the table schema  |
